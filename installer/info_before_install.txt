CalmWeb - System-wide Web Protection

CalmWeb is a web proxy filter designed to protect users from ads, scams,
and remote control software. It works system-wide with any browser.

Features:
- Blocks known scam and malicious websites
- Blocks remote desktop/assistance software (TeamViewer, AnyDesk, etc.)
- Blocks ads and tracking domains
- Blocks direct IP address access (prevents common scam techniques)
- Blocks non-secure HTTP traffic by default
- Auto-updates blocklists from trusted community sources

CalmWeb runs as a local proxy on port 8080 and configures the Windows
system proxy to route all web traffic through it.

Requirements:
- Windows 10 or Windows 11
- Administrator privileges
- Internet connection for blocklist downloads
